## 🇨🇳 中文

### 📖 项目简介
欢迎来到 **X Space** 小组的开源仓库！本项目为 **ENT208TC** 课程的团队项目 —— **On Computer Voice Assistant (桌面语音助手)**。

我们的目标是探索人机交互的边界，利用先进的自然语言处理 (NLP) 与语音识别算法，开发一款便捷、智能的计算机本地语音助手，实现流畅的语音交互与本地环境的指令执行。
* **课程名称**: ENT208TC
* **项目名称**: On-computer Voice Assistant 
* **小组编号**: Session 2 - Group 1

---

### ⚙️ 硬件设备
本项目结合了实体硬件以增强交互反馈与音频处理能力：
* **核心控制设备:** M5Stick Atom S3 R
* **音频扩展模块:** Echo Base

本仓库致力于透明、完整地记录我们项目的全生命周期：

- 💻 **`Software Implementation/`** *(🚧 To be continued...)* - 将存放软件功能的架构设计与核心代码，包括语音处理模块与指令执行模块。

### ✨ 功能特性 (开发中)
- [ ] **语音唤醒与精准识别**：快速响应用户的语音指令。
- [ ] **意图理解 (NLP)**：准确解析用户需求并转化为可执行的操作。
- [ ] **本地指令执行**：控制计算机系统状态与应用软件。
- [ ] **硬件设备联动**：利用 **M5Stick Atom S3 R + Echo Base** 实现物理交互反馈及音频输入/输出。
